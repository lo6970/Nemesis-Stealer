# NEMESIS

Stealer Builder & Windows Toolkit

---

## What you need

- Windows 10 or 11
- Python 3.10 or newer installed
- The ZIP file from Releases

---

## How to install

1. Download the ZIP from Releases
2. Extract everything to a folder
3. Double click `setup.bat` and wait for it to finish
4. Start `Nemesis.exe`

The setup.bat installs CustomTkinter, PyInstaller, Pillow, OpenCV, and Requests automatically. You need internet connection for this. The installation only needs to be done once.

---

## How to use

When you open Nemesis.exe you see a menu with 9 numbered options. Type the number and press Enter to select something. After each tool finishes, press Enter to go back to the menu. Type 9 and Enter to close the program.

---

## Menu Options

**1 - Build Stealer** opens a graphical window where you can build your own executable. Enter a Discord webhook URL and click Test to make sure it works. Select which modules you want by checking the boxes. Enter a name for your output file. Click BUILD EXE and wait. The finished file appears in the build folder. When someone runs your generated file, it sends a preview message with PC info to your webhook, then uploads a ZIP with all collected data.

**2 - Username Search** checks if a name exists on 20 platforms like GitHub, Twitter, Instagram, TikTok, Reddit, YouTube, Steam, Spotify and more. Green FOUND means the account exists, red NOT means it does not.

**3 - IP Lookup** shows the country, city, provider, and ISP of any IP address you enter.

**4 - Email Breach** checks if an email appears in known data breaches and lists which services were compromised.

**5 - Phone Lookup** validates a phone number and shows the country, carrier, and line type.

**6 - DNS Lookup** finds the IP addresses of any domain name.

**7 - Webhook Info** shows details about a Discord webhook URL including its name, channel, and server.

**8 - Port Scanner** checks 15 common ports on an IP address. Green is open, red is closed.

**9 - Exit** closes the program.

---

## Stealer Modules

The builder includes Screenshot, Camera, Discord Token, Browser Passwords from Chrome and other browsers, Browser History, Cookies, Credit Cards, System Info, WiFi Passwords, Installed Programs, and a custom Popup message.
